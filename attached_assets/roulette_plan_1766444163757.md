# Safe + Steady + Spicy Roulette Plan (00) — Quick Ops Doc (v4)

**Assumption:** American roulette **0 + 00** (avoid **000** if you care about value).  
**Unit:** `U = $5`

---

## Session Limits (recommended)

- **Stop-win:** `+8U` = **+$40**
- **Stop-loss:** `-10U` = **-$50**
- **No martingale.** Never increase bets after losses.

---

## Base Mode (MOST spins): 2 bets (2U = $10)

Every Base spin, place:

1. **BET 1 (Anchor):** `1U` on the **Current Anchor** (10-spin block rotation)
2. **BET 2 (Mix bet):** `1U` on **either a Dozen OR a Column** (chosen “randomly” per 10-spin block)

### BET 1 — Anchor (same as before): 10-spin block rotation
Anchor changes only at the start of each **10-spin block** (spins 1–10, 11–20, ...):

Block cycle:
1. **BLACK**
2. **ODD**
3. **RED**
4. **EVEN**
5. **1–18**
6. **19–36**
→ repeat

**2-loss flip rule (within the block):** if the anchor loses **2 spins in a row**, flip to its pair for the rest of the block:
- BLACK ↔ RED
- ODD ↔ EVEN
- 1–18 ↔ 19–36

### BET 2 — “Random” Dozen/Column per 10-spin block (no clock needed)
At the start of each 10-spin block, choose BET 2 using the **last result number** as a seed (this is for randomness only; not prediction).

**Seed value `S`:**
- If last result is `00`, set `S = 37`
- If last result is `0`, set `S = 0`
- Otherwise `S = (the number)`

**Step A — pick Dozen vs Column**
- If `S` is **odd** → BET 2 is a **Dozen**
- If `S` is **even** → BET 2 is a **Column**

**Step B — pick which Dozen/Column (use `S mod 3`)**
- If BET 2 is **Dozen**:
  - `S mod 3 = 0` → **1–12**
  - `S mod 3 = 1` → **13–24**
  - `S mod 3 = 2` → **25–36**
- If BET 2 is **Column**:
  - `S mod 3 = 0` → **1st column**
  - `S mod 3 = 1` → **2nd column**
  - `S mod 3 = 2` → **3rd column**

**Rule:** do **not** change BET 2 mid-block.

## Party Mode (OCCASIONAL): add 1 corner bet (3U = $15)

Party Mode is **1 spin only** when called by the assistant (max **1 Party spin per 10-spin block**).

Party Mode is allowed if **either**:
- Session P/L is **≥ +4U** (**+$20**) **OR**
- The most recent result seed `S` satisfies **`S mod 7 = 0`** (random-ish trigger)

When Party Mode is ON for a spin, place Base Mode bets **plus**:

3. **BET 3 (Corner):** `1U` on the **Current Party Corner** (rotation below)

### Party corner rotation (fixed list)
Rotate through this list each time Party Mode is called:
- **C1:** 26–27–29–30
- **C2:** 14–15–17–18
- **C3:** 8–9–11–12
- **C4:** 2–3–5–6
- **C5:** 20–21–23–24
- **C6:** 32–33–35–36
→ repeat
